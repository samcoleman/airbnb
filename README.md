# 37 Kings Road Airbnb Captive Portal

UniFi captive portal for guest network.

## Setup

```bash
npm install
cp site/backend/.env.example site/backend/.env
# Edit .env and run:
npm run dev
```

## Scripts

- `npm run dev` - Development
- `npm run build` - Build
- `npm run docker:up` - Docker
- `npm run docker:down` - Stop Docker
